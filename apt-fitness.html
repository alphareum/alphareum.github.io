Project Overview

APT Fitness Assistant is a comprehensive AI-powered fitness application that combines computer vision, machine learning, and scientific body composition analysis to provide personalized fitness guidance. The system uses advanced pose tracking to analyze body composition from photos and generates customized workout recommendations based on individual goals and capabilities.

Computer Vision Analysis

Advanced pose detection using MediaPipe to extract body measurements from photos with sub-pixel accuracy

AI-Powered Recommendations

Machine learning algorithms generate personalized workout plans based on your goals, fitness level, and equipment

Progress Tracking

Comprehensive analytics dashboard to monitor your fitness journey with visual progress charts and metrics

Implementation Highlights

Enhanced Computer Vision Pipeline

The system uses a multi-stage image processing pipeline with adaptive enhancement based on image quality assessment. This includes:

  • Multi-model pose detection with intelligent fallback
  • Sub-pixel accuracy for landmark positioning
  • Perspective and lens distortion correction
  • Confidence-weighted measurement extraction

Scientific Body Composition Methods

Multiple validated formulas are combined with dynamic weighting based on individual characteristics: 

# Navy Method Body Fat Formula (±3-4% accuracy)
if gender == 'male':
    body_fat = 495 / (1.0324 - 0.19077 * log10(waist - neck) + 
                     0.15456 * log10(height)) - 450
else:
    body_fat = 495 / (1.29579 - 0.35004 * log10(waist + hip - neck) + 
                     0.22100 * log10(height)) - 450

Machine Learning Integration

An ensemble of ML models provides enhanced accuracy for body composition estimation:

  • Random Forest, Gradient Boosting, and Ridge Regression models
  • 13 anthropometric features for comprehensive analysis
  • 5000+ synthetic training samples with realistic correlations
  • Robust scaling for outlier handling
