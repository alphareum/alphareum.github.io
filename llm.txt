# Alvin's Professional Portfolio Website

## Project Overview
This is a professional portfolio website for Alvin, a professional researcher and financial market analyst. The website is hosted on GitHub Pages at `alphareum.github.io` and showcases his research work, scriptwriting portfolio, and technical projects.

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Grid and Flexbox
- **Fonts**: Inter font family from Google Fonts
- **Icons**: Font Awesome 6.4.0
- **Deployment**: GitHub Pages with automated deployment workflow
- **Domain**: alphareum.github.io

## Site Structure

### Main Pages
- `index.html` - Homepage with full portfolio overview
- `ailerts-automation.html` - Detailed project page for video processing system
- `apt-fitness.html` - Detailed project page for AI fitness assistant
- `backup.html` - Backup version of homepage (includes research section)
- `sample.html` - Sample version with "coming soon" placeholder

### Assets
- `styles.css` - Main stylesheet with utility classes
- `script.js` - JavaScript functionality for interactions
- `.github/workflows/` - GitHub Actions for automated deployment
